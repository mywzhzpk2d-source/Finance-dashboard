# BJ Market Terminal v2

개인용 시장 분석 대시보드 프로토타입.

## 구성
- Macro 분석
  - 미국 10년물/2년물 금리
  - 10Y-2Y 스프레드 영역
  - DXY / M2 / Gold / WTI / Copper
  - NASDAQ / Dow Jones / KOSPI / VIX
  - 시장강도 순위
  - 거래대금 순위
- 매매일지
- 섹터별 관심종목
- AI Analyst 프롬프트 생성기

## 실행
압축 해제 후 index.html 실행.

## 주의
현재 차트는 UI 검증용 샘플 데이터입니다.
실시간 데이터 연결은 별도 API 연동 단계에서 구현합니다.
