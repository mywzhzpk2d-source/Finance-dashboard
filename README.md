# BJ Finance Terminal

개인용 금융·투자 대시보드 첫 버전입니다.

## 실행 방법
1. 폴더의 `index.html` 파일을 더블클릭합니다.
2. 브라우저에서 바로 실행됩니다.
3. 관심종목, 매매일지, 시장 메모는 브라우저의 localStorage에 저장됩니다.

## 포함 기능
- 개인 시장 대시보드
- 일일 시장 체크리스트
- 시장 메모
- 관심종목 관리
- 매매일지
- 금융공학 공부 로드맵
- 개인 투자 성향 기반 AI 분석 프롬프트 생성기
- 다크 모드
- 모바일 반응형
